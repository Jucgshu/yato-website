Site Web du Groupe YATO 🎸

Site officiel du groupe de reprises rock YATO - Un univers musical où l'énergie rock rencontre l'élégance numérique.
🌟 À propos

Bienvenue sur le site web officiel de YATO, groupe de reprises rock passionné qui enflamme les scènes depuis 1999. Ce site présente notre actualité, nos concerts, notre galerie et offre un moyen direct de nous contacter.
🚀 Fonctionnalités clés

    Expérience immersive avec bannière animée et effets visuels

    Agenda des concerts dynamique avec filtres intelligents

    Galerie photos interactive avec lightbox avancée

    Section vidéos intégrant nos meilleures performances

    Formulaire de contact avec double système d'envoi

    Design responsive parfaitement adapté à tous les appareils

    Système de notification pour les annonces importantes
